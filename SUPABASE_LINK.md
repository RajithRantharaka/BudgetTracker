# Supabase Dashboard Access

To set up your database and get your API keys as described in `DEPLOYMENT.md`:

**Supabase Dashboard:** [https://supabase.com/dashboard/sign-in](https://supabase.com/dashboard/sign-in)

Once logged in:
1.  Create a **New Project**.
2.  Go to **SQL Editor** to run the schema script.
3.  Go to **Settings > API** to get your Keys (`URL` and `anon key`).
